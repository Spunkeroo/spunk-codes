Available Endpoints

    
A sample of our most popular API endpoints. Over 50 endpoints available across all tool categories.

  


  

    

      POST
      
/api/v1/json/format

      
Format, validate, and minify JSON. Supports large payloads up to 10MB.

    

    

      GET
      
/api/v1/color/palette

      
Generate color palettes. Parameters: base color, harmony type, count.

    

    

      POST
      
/api/v1/seo/meta-tags

      
Generate optimized meta tags from URL or content. Returns OG, Twitter, and standard tags.

    

    

      GET
      
/api/v1/security/password

      
Generate cryptographically secure passwords. Configurable length, complexity, and character sets.

    

    

      POST
      
/api/v1/dev/regex-test

      
Test regex patterns against input strings. Returns matches, groups, and execution time.

    

    

      POST
      
/api/v1/content/markdown

      
Convert Markdown to HTML and vice versa. Supports GFM, tables, and code highlighting.

    

    

      GET
      
/api/v1/dev/uuid

      
Generate UUIDs (v1, v4, v5). Bulk generation supported with count parameter.

    

    

      POST
      
/api/v1/seo/schema

      
Generate Schema.org structured data. Supports Article, Product, FAQ, HowTo, and more.
